**roi**

---

### 1. Hours Saved

`hours_saved = time_required_using_alternative_method − time_required_using_the_product`

Where the alternative method can be:

* Excel
* handwritten logging
* manual data entry
* manual analysis / reconciliation

So this is **not** the same as tracked usage time. It is the **delta** between the old workflow and the new workflow.

If measured across multiple tasks:

`hours_saved = Σ(baseline_task_time − product_task_time)`

---

### 2. ROI

`roi = (hours_saved × hourly_rate) − subscription_cost`

---

### 3. Plan Cost

`plan_cost = subscription_price`

---

### 4. Payback Period

`payback_period_hours = subscription_cost ÷ hourly_rate`

---

### Final equation summary

`roi = ((baseline_hours − product_hours) × hourly_rate) − plan_cost`

or, per task:

`roi = (Σ(task_baseline_time − task_product_time) × hourly_rate) − plan_cost`
